# 230323_수업내용

< 클래스와 함수 >

- 함수 만들기
- FirstIn, LastOut (Stack형 구조)
- FirstOut (큐형 구조)
- 재귀 함수를 만들 때는, 종료 조건을 반드시 명시해야 한다.
- 배열이 꼭 3x4 같은 모양이 아닐 수도 있다.
- 배열 모양이 계단모양, ㄷ모양 등등이 가능함.
- int[ ][ ] array = new int [ 3 ][ 4 ]
array [ 0 ][ 0 ] = 1;
array [ 0 ][ 1 ] = 2;
array [ 0 ][ 2 ] = 3;
array [ 0 ][ 3 ] = 4;
- 변수에 일반 변수[값 저장]와 참조 변수[주소지 저장]가 있다.
- 배열을 new [3]해서 만들면, 내부 값들은 다 0으로 초기화됨.
